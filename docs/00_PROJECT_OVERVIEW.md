# MVO Inventory Management System

Version: 0.9 (Draft)

---

# Project Overview

MVO Inventory Management System — це корпоративна вебсистема для централізованого обліку матеріальних цінностей, що знаходяться на відповідальному зберіганні матеріально відповідальних осіб (МВО).

Основна мета проєкту — повністю замінити паперовий облік майна сучасною електронною системою з повною історією руху кожної одиниці майна.

---

# Mission

Створити просту, швидку та надійну систему обліку матеріальних цінностей, яка:

- мінімізує людські помилки;
- забезпечує повний аудит усіх операцій;
- виключає втрату історії руху майна;
- працює однаково швидко для будь-якої кількості користувачів;
- легко масштабується.

---

# Project Goals

Система повинна забезпечувати:

- централізований облік залишків;
- імпорт майна з бухгалтерських систем;
- передачу майна між МВО;
- журнал усіх операцій;
- аудит усіх дій користувачів;
- контроль доступу за ролями;
- формування звітів;
- можливість подальшої інтеграції з іншими інформаційними системами.

---

# Target Users

Система розрахована на:

- Owner
- Auditor
- DPP Administrator
- Materially Responsible Person (МВО)

У перспективі можуть бути додані додаткові ролі.

---

# Main Principles

Проєкт будується навколо декількох базових принципів.

## 1. Дані важливіші за інтерфейс

Інтерфейс може змінюватися.

Структура даних повинна залишатися стабільною.

---

## 2. Усі операції повинні бути відтворюваними

Будь-яку зміну необхідно мати можливість пояснити.

Жодна операція не повинна зникати безслідно.

---

## 3. Мінімум ручної роботи

Якщо процес можна автоматизувати — він має бути автоматизований.

---

## 4. Безпека за замовчуванням

Кожна нова функція повинна бути безпечною ще до її публікації.

---

## 5. Простота

Інтерфейс повинен бути максимально простим навіть для користувача без технічної підготовки.
